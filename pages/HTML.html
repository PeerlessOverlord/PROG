Elements (tags) you can use for HTML documents

Non void Elements that you would use for text:

  • h1 - h6
  • p
  • code
  • em
  • strong

big elements most other elements are put inside:

  • head
  • body
  • html

deprecated elements, these should be avoided or removed from documents:

  • abbr
  • applet
  • basefont
  • big
  • center
  • dir
  • font
  • frame
  • frameset
  • noframes
  • strike
  • tt
